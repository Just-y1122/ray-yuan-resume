GitHub Pages 部署步骤：
1. 先解压本压缩包。
2. 打开 GitHub 仓库 Just-y1122/ray-yuan-resume。
3. 点击 Add file → Upload files。
4. 把解压后的所有文件和 assets 文件夹一起上传到仓库根目录。
   注意：不要直接上传 ZIP 压缩包，也不要再套一层文件夹。
5. 提交后进入 Settings → Pages。
6. Build and deployment 的 Source 选择 Deploy from a branch。
7. Branch 选择 main，文件夹选择 /(root)，然后点击 Save。
8. 等待约几分钟后访问：https://just-y1122.github.io/ray-yuan-resume/
